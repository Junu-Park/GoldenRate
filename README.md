# 금니 (GoldenRate)
> 예적금 금리 정보를 한눈에, 자신에게 맞는 예적금 상품을 쉽게 찾을 수 있는 iOS 앱

## 📱 소개
**금니**에서는 최근 기준금리, 1금융권과 2금융권 예적금 금리 정보를 차트로 한눈에 볼 수 있고,  
원하는 조건에 맞는 예적금 상품을 쉽게 찾을 수 있는 앱입니다.

- ⭐️ 최근 3가지 금리(기준금리, 1금융, 2금융) 차트
- ⭐️ 예적금 상품 TOP3 랭킹
- ⭐️ 예적금 상품 필터를 통한 맞춤 검색
- ⭐️ 예적금 이율 계산기
- ⭐️ 최신 금리 정보 제공 위젯
